Overview

Grim Fate Awaits was the 3rd game I worked on at TGA, with a duration of 12 weeks. The game was made in TGE (The Game Engine), TGAs in-house game engine, with Unity acting as a level designer using an export script that exports scenes into JSON files.

My main contributions in this project was the player movement and the falling spikes

Player Movement

I was in charge of implementing the player movement and matching it as close as possoble to Hollow Knight, our reference game, using the metrics set by our level designers.
